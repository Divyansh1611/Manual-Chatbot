How to Run

1. Backend:
    1.1) Save the backend code in chatbot.py.
    1.2) Install Flask and Flask-Cors: (Run below command to install these libraries) -- run only one time.
            pip install flask flask-cors

2. Start the backend server: (Run the below command into the terminal so server runs and data is fetched by index.html file)
    
    python chatbot.py


3. Frontend:
    3.1) Save the frontend code in index.html.
    3.2) Open index.html in any modern web browser(But only after 1 and 2nd product ).
